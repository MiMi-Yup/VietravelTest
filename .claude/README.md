# README - Phân tích bài toán

## 1. Hiểu bài toán
- Hệ thống tập trung vào việc quản lý các yêu cầu đặt dịch vụ tour với các ràng buộc chặt chẽ về ngân sách và loại hình tour
- Mục tiêu là tự động hóa việc tính toán và phân luồng phê duyệt dựa trên giá trị đơn hàng

## 2. Các giả định nghiệp vụ 
- Giả định rằng "Đơn giá" trong các dịch vụ đã bao gồm các loại thuế phí liên quan.
- Giả định rằng "Người phụ trách" là nhân viên hiện tại đang đăng nhập vào hệ thống.
